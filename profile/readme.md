# HVACoder.com

HVACoder develops engineering software systems, standards, and documentation for structured and traceable HVAC/MEP design workflows.

Our focus is on transforming workflows—often spread across spreadsheets, notes, and tribal knowledge—into structured, governed, and software-driven processes.

## Approach

- formalized engineering data and domain models
- explicit, traceable design decisions
- documentation as part of the system—not separate from it
- repository-driven development with auditable history

## Current Focus

### DesignBench

DesignBench is an active project focused on formalizing HVAC/MEP design data, decisions, and workflows into a structured system that supports consistency, transparency, and long-term maintainability.
